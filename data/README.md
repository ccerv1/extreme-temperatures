# Data Directory

## stations.json

Registry of ~50 major US city weather stations with GHCN station IDs, coordinates, and metadata. Used by the backfill pipeline and frontend.

## DuckDB Database

The `extreme_temps.duckdb` file (gitignored) is generated by the backfill pipeline and contains:

| Table | Description |
|-------|-------------|
| `dim_station` | Station metadata and coverage stats |
| `fact_station_day` | Daily observations (tmin_c, tmax_c, tavg_c, prcp_mm) |
| `fact_station_window_recent` | Precomputed rolling window aggregates |
| `dim_climatology_quantiles` | Percentile distributions by day-of-year |
| `dim_station_records` | All-time records per window size |
| `fact_station_latest_insight` | Precomputed latest insights for home page |

Temperatures are stored in Celsius internally.
